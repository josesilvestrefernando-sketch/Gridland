# Gridland
A simple sandbox-style simulation inspired by the game WorldBox. Built with the Processing library and designed to run on Processing for Android.

This project is a small sandbox-style simulation inspired by the game WorldBox. The goal of this project is to experiment with simple world simulation mechanics using the Processing library.

The application is designed to run using Processing Android mode, allowing the sketch to be exported and tested on Android devices.

This project was created as a learning exercise to explore:

Basic world simulation concepts

Grid or tile-based environments

Simple interaction mechanics

Mobile deployment using Processing for Android

The project is only a simple prototype/demo and is not intended to replicate the full features of the original game.

Technologies Used

Java (Processing language)

Processing

Processing for Android
